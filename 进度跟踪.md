## 关键设计决策

| 决策 | 选择 | 原因 |
|------|------|------|
| AI客户端 | openai npm SDK | 原生支持自定义baseURL |
| 格式适配 | 独立适配层（adapters/） | 解耦，易测试 |
| 配置存储 | `~/.tikatak-codex/settings.json` + `apikey` 分离 | API Key 权限0600 |
| 工具框架 | 全新实现 | 原框架依赖 bun，不可移植 |
| TUI框架 | React + Ink | 与原项目一致 |
| 重试策略 | 指数退避（withRetry.ts） | 429/5xx 自动重试 |
| SubAgent防递归 | SUB_AGENT_TOOL_SCHEMAS（排除SubAgent自身） | 防止无限递归 |
| ESM+CJS兼容 | createRequire banner shim | commander 等 CJS 包在 ESM bundle 中正常运行 |

## 关键设计决策（已落地）

| 决策 | 选择 | 原因 |
|------|------|------|
| AI客户端 | openai npm SDK | 原生支持自定义baseURL，覆盖所有OAI兼容提供商 |
| 格式适配 | 独立适配层（adapters/） | 解耦，易测试，不污染工具/命令层 |
| 配置存储 | `~/.tikatak-codex/settings.json` + `apikey` 分离 | API Key单独文件，权限0600，不混入配置 |
| 工具框架 | 全新实现（不复用原框架） | 原框架依赖bun/Anthropic SDK/内部100+文件，不可移植 |
| TUI框架 | React + Ink（与原项目一致） | 无缝继承原项目UI风格 |
| 重试策略 | 指数退避（withRetry.ts） | 429/5xx自动重试，Retry-After头支持 |
