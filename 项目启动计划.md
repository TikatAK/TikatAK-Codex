# TikatAK-Codex 项目启动计划

> 项目目标：基于 claude-code 2.1.88 源码，构建支持任意 OpenAI 兼容后端的开源编码 AI 工具
> 工期单位：AI 工时（约等于连续生成时间，非人类工作日）

---

## 项目定位

**TikatAK-Codex** 是一个从 Claude Code 派生的、面向开发者的 AI 编码助手 CLI 工具，核心差异：

- 🔓 **不绑定 Anthropic**：支持 OpenAI、DeepSeek、Kimi、Qwen、本地模型等任意兼容后端
- 🔑 **自带 API Key**：用户自行配置提供商和密钥，无需 claude.ai 账号
- 🛠️ **保留全部编码能力**：Bash、FileEdit、Grep、多Agent、MCP、Git集成等全部保留
- 🪶 **去除非编码功能**：移除 OAuth、Computer Use、Files API 等与写代码无关的模块

### Phase 1 — 提供商抽象层（AI工时：6h）

**目标**：实现核心 Provider 框架，让项目可以连接任意 OpenAI 兼容端点

- [ ] 新建 `src/providers/types.ts`：定义 `ProviderConfig` 接口

- [ ] 修改 `utils/auth.ts`：新增 provider API Key 的存读删逻辑（跨平台：keychain/config文件）
- [ ] 修改 `settings/settings.ts`：新增 `provider` 配置 schema
- [ ] 新建 `commands/provider.ts`：`/provider set|list|test|status` 命令实现
- [ ] 新建 `utils/providerPresets.ts`：常用提供商端点预置表

**产出**：可配置提供商，可存储和读取 API Key

---

### Phase 2 — 消息格式适配层（AI工时：8h）

**目标**：实现 Anthropic Messages API ↔ OpenAI Chat Completions API 双向转换

- [ ] 新建 `adapters/openai/requestAdapter.ts`：Anthropic 请求格式 → OpenAI 请求格式
  - system prompt 转换
  - 多轮消息 role 映射
  - 工具定义格式转换（tool_use → function schema）
- [ ] 新建 `adapters/openai/responseAdapter.ts`：OpenAI 响应 → Anthropic 响应格式
  - content 结构重建
  - stop_reason 映射（stop/tool_calls → end_turn/tool_use）
  - usage 字段映射
- [ ] 新建 `adapters/openai/streamAdapter.ts`：OpenAI SSE chunks → Anthropic stream events
  - content_block_start / content_block_delta 重建
  - tool_calls streaming 转 tool_use streaming
- [ ] 新建 `adapters/openai/toolAdapter.ts`：工具调用双向适配
  - parallel tool calls 处理
  - tool result 消息格式转换
- [ ] 集成适配层到 `services/api/client.ts`
- [ ] 端到端测试：发一条带工具调用的消息，验证完整流程

**产出**：工具调用和流式输出完全正常工作

## 技术栈

| 项目 | 选择 | 说明 |
|------|------|------|
| 语言 | TypeScript | 与原项目一致 |
| AI 客户端 | `openai` npm 包 | 官方 SDK，支持自定义 baseURL |

| 配置存储 | `~/.tikatak/settings.json` | 跨平台，替代 `~/.claude/` |
| 依赖管理 | npm | 与原项目一致 |

## 风险提示

| 风险 | 等级 | 应对 |
|------|------|------|
| 部分第三方 OpenAI 兼容 API 工具调用实现不完整 | 中 | 加入 provider 兼容性标记，提示用户 |
| 不同模型遵循指令能力差异大 | 中 | 推荐使用 GPT-4o 或 DeepSeek-V3 |
| 原始源码为非官方还原，可能有细节缺失 | 低 | 逐模块测试，缺失处补全 |
| 与原始 claude-code 版本升级脱节 | 低 | 此项目独立维护，不追随版本 |
